# codealpha_tasks-
age calculator
 JavaScript Age Calculator 
A simple and responsive Age Calculator built using **HTML, CSS, and JavaScript**.  
It calculates the exact age (years, months, and days) from the user’s date of birth.
 Features
- Select date of birth using date picker
- Prevents selecting future dates
- Calculates accurate age in:
  - Years
  - Months
  - Days
- Clean and minimal UI
- Responsive layout
- Beginner-friendly JavaScript logic
 Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling & layout
- **JavaScript (ES6)** – Logic & DOM manipulation
